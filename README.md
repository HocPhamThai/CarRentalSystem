# CarRentalSystem

To run app you need to configure your Dbconnection string after that:

-   You need to import database for this app in file "CarRentalSystem.sql".
-   You need to install visual studio or visual studio code to run app,

Some requirements:
.NET runtime, SQL server, IDE

## Login Form

-   Admin: Users, Cars, Customers, Bookings, Schedules, Dashboard
-   Employee: Cars, Customers, Bookings, Schedules

## Main Form

-   Navigation Panel: Users, Cars, Customers, Bookings, Schedules, Dashboard
-   Content Panel: Displays the selected module from the navigation panel Based on user role (Admin or Employee).
